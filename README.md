## Description
The following files are added to the MESSAGEix Westeros tutorial:
- `westeros_baseline_gas.ipynb` modifies the baseline tutorial to include gas power plants.
- `westeros__emissions_taxes_gas.ipynb` introduces emission taxes.

Please run the new files in their respective order.
